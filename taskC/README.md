## 1. Подробное объяснение вопроса

На территории поселка N есть три базовые станции МТС, каждая из которых покрывает круговую зону радиусом `a`, `b` и `c` соответственно.

Условие измерений сигналов следующее:

* Есть ровно одна точка, где принимается сигнал от базовых станций 1 и 2.
* Есть ровно одна точка, где принимается сигнал от базовых станций 2 и 3.
* Есть ровно одна точка, где принимается сигнал от базовых станций 1 и 3.

Необходимо определить **минимальное и максимальное значение квадрата площади треугольника**, образованного тремя базовыми станциями.

Если ситуация невозможна (треугольник не существует), выводится `-1`.

Пример:

```
Вход:
1
1
1

Выход:
3 3
```

---

## 2. Теоретическое объяснение решения

1. **Смысл задачи:**
   Три базовые станции образуют треугольник. Известны радиусы покрытия, и даны условия пересечения. Эти пересечения дают возможные расстояния между станциями.

2. **Вывод возможных расстояний:**
   Пусть три станции имеют расстояния между собой `d01`, `d02`, `d12`. Используем известные радиусы для вычисления этих расстояний:

   * `d01` (между станциями 1 и 2) может быть `a+b` или `|a-b|`.
   * `d02` (между станциями 1 и 3) может быть `a+c` или `|a-c|`.
   * `d12` (между станциями 2 и 3) может быть `b+c` или `|b-c|`.

   Таким образом, формируем несколько кандидатов комбинаций расстояний, проверяя все варианты.

3. **Проверка существования треугольника:**
   Для треугольника с длинами сторон `x`, `y`, `z` должно выполняться неравенство треугольника:

   ```
   ```

x + y > z
x + z > y
y + z > x

```

4. **Вычисление площади:**
   Используем **формулу Герона** для вычисления площади треугольника:
```

S = √(s * (s - a) * (s - b) * (s - c))
s = (a + b + c)/2

````
   Так как требуется квадрат площади, используем `S² = s * (s - a) * (s - b) * (s - c)`.

5. **Целостность результата:**
   Проверяем, чтобы квадрат площади был целым числом, как указано в задаче.

## 3. Подробный разбор кода решения

```python
#!/usr/bin/env python3
import sys
from fractions import Fraction

# Функция вычисления квадрата площади треугольника по формуле Герона
def heron_sq(a, b, c):
    # Проверка на существование треугольника
    if not (a+b>c and a+c>b and b+c>a):
        return 0
    s = Fraction(a+b+c, 2)  # полупериметр
    area_sq = s*(s-a)*(s-b)*(s-c)  # квадрат площади
    return area_sq if area_sq > 0 else 0

def main():
    # Чтение входных данных
    data = sys.stdin.read().strip().split()
    if len(data) < 3:
        return
    a, b, c = map(int, data)
    
    # Возможные комбинации расстояний между станциями
    candidates = [
        (a+b, a+c, b+c),
        (abs(a-b), a+c, b+c),
        (a+b, abs(a-c), b+c),
        (a+b, a+c, abs(b-c))
    ]
    
    res = []
    for d01, d02, d12 in candidates:
        if 0 in (d01, d02, d12):  # исключаем совпадение точек
            continue
        area_sq = heron_sq(d01, d02, d12)  # вычисление квадрата площади
        if area_sq and area_sq.denominator == 1:  # проверка целого числа
            res.append(area_sq.numerator)
    
    # Вывод результата
    if not res:
        print(-1)
    else:
        print(min(res), max(res))

if __name__ == "__main__":
    main()
````

### Объяснение кода:

1. `heron_sq(a, b, c)` — функция вычисляет **квадрат площади** треугольника по формуле Герона, используя `Fraction` для точных вычислений.
2. Чтение входных данных производится через `sys.stdin.read()`.
3. `candidates` — список всех возможных комбинаций расстояний между станциями с учётом радиусов покрытия.
4. Для каждой комбинации проверяем:

   * расстояния не равны нулю;
   * существует треугольник;
   * квадрат площади — целое число.
5. Если найдены подходящие комбинации, выводим минимальное и максимальное значение квадрата площади. В противном случае выводим `-1`.
