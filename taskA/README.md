## 1. Подробное объяснение вопроса

В задаче описана система cashback для клиентов МТС Банка:  

- Если покупка составляет **не менее `a` рублей**, клиент получает фиксированный кешбэк **`b` рублей** в начале следующего месяца.  
- В августе клиент потратил **`x` рублей**.  

Необходимо определить, какой **максимальный кешбэк** он может получить на свой счет в начале сентября.  

Примеры входных данных:

- `a = 100` (минимальная стоимость покупки)  
- `b = 1` (сумма кешбэка за одну покупку)  
- `x = 2025` (потраченная сумма)  

**Максимальный кешбэк** достигается при **максимальном числе покупок стоимостью ≥ a**, которые можно совершить за общую сумму `x`.

---

## 2. Теоретическое объяснение решения

1. **Определение количества покупок**:  
   Любая покупка ≥ `a` рублей приносит `b` рублей кешбэка.  

2. **Вычисление максимального числа таких покупок**:  
   \[
   n = \left\lfloor \frac{x}{a} \right\rfloor
   \]  
   где `n` — максимальное число покупок, которые можно совершить за потраченную сумму.

3. **Вычисление кешбэка**:  
   \[
   \text{cashback} = n \cdot b
   \]

Пример:  
- `x = 2025`, `a = 100` → `2025 // 100 = 20` покупок  
- Кешбэк: `20 * 1 = 20` рублей  

---

## 3. Подробный разбор кода решения

```python
# Чтение входных данных
a = int(input())  # минимальная стоимость покупки
b = int(input())  # сумма кешбэка за одну покупку
x = int(input())  # потраченная сумма в августе

# Вычисление максимального числа покупок стоимостью ≥ a
n = x // a  # целочисленное деление

# Вычисление максимального кешбэка
cashback = n * b

# Вывод результата
print(cashback)
```

**Пошаговое объяснение кода:**  

1. `a = int(input())` — минимальная стоимость покупки.  
2. `b = int(input())` — сумма кешбэка за одну покупку ≥ `a`.  
3. `x = int(input())` — общая потраченная сумма за август.  
4. `n = x // a` — максимальное количество покупок, которые можно сделать.  
5. `cashback = n * b` — общий кешбэк.  
6. `print(cashback)` — вывод результата.  

**Особенности решения:**  
- Короткий и эффективный код.  
- Использует только базовые операции.  
- Работает для всех допустимых входных данных.
