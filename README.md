## Описание раит-апа

Каждая задача в этом раит-апе представлена в отдельной папке и содержит два файла:  

1. **задача (PDF)** — полная формулировка задачи.  
2. **Решение (PY)** — файл с готовым кодом решения.  

Каждая задача разбита на три раздела:  

1. **Подробное объяснение вопроса** — разбор всех условий и требований задачи.  
2. **Теоретическое объяснение решения** — пошаговое объяснение логики и методов, необходимых для решения.  
3. **Подробный разбор кода решения** — объяснение каждой строки кода и используемых конструкций.  

Такой подход позволяет полностью понять как саму задачу, так и методы её решения на практике.

## Задачи

### Задача A. Максимальный кешбэк
Клиенты МТС Банка получают фиксированный кешбэк за покупки свыше определённой суммы. Необходимо вычислить **максимальный кешбэк**, который мог получить клиент за месяц.  
**Входные данные:** стандартный ввод.  
**Выходные данные:** стандартный вывод.  

### Задача B. Телефонные номера
Дана строка цифр. Нужно проверить, можно ли её полностью разделить на **корректные российские мобильные номера**.  
**Входные данные:** стандартный ввод.  
**Выходные данные:** стандартный вывод.  

### Задача C. Сигнал от базовых станций
На территории посёлка есть три базовые станции с заданными радиусами покрытия. Необходимо найти **минимальное и максимальное значение квадрата площади треугольника**, образованного станциями, при описанных условиях.  
**Входные данные:** стандартный ввод.  
**Выходные данные:** стандартный вывод.
